R with RStudio: getting started

---

A short introduction to R using RStudio. Hands on. No previous experience required.

---

<p>
	R is a popular programming language useful for statistics and data science.
</p>
<p>
	The UQ Digital Scholars Hub (DHS) is offering a free RStudio introductory, hands-on session for R beginners.
</p>
<p>
	What this session will cover:
</p>
<ul>
	<li>  Create a project for data analysis</li>
	<li>Introduce useful functions</li>
	<li>Working with scripts</li>
	<li>Where to find help</li>
	<li>Import data</li>
	<li>Use a few shortcuts</li>
	<li>Install and load extra packages</li>
	<li>An example of data visualisation</li>
</ul>
<p>
	The software is available on the training room's computers. If you prefer to use your own laptop, you are also welcome to bring it in (but please install the software beforehand, or come 20 minutes earlier if you need to install the software).
</p>
