R with RStudio: getting started

---

A short introduction to R using RStudio. Hands on. No previous experience required.

---

The UQ Digital Scholars Hub (DHS) is offering a free RStudio introductory, hands-on session for R beginners.

What this session will cover:

    Create a project for data analysis
    Introduce useful functions
    Working with scripts
    Where to find help
    Import data
    Use a few shortcuts
    Install and load extra packages
    An example of data visualisation

The software is available on the training room's computers. If you prefer to use your own laptop, you are also welcome to bring it in (but please install the software beforehand, or come 20 minutes earlier if you need to install the software).

To install R and RStudio on your own laptop, refer to the instructions:
https://gitlab.com/stragu/CDS/blob/master/R/Installation.md

If you are interested in other R sessions, please search for "RStudio" on StudentHub or look at the list https://studenthub.uq.edu.au/students/events/search?query=rstudio
