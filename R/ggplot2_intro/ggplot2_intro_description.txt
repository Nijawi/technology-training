This description is for the StudentHub event form

--- Title of session

R data visualisation with RStudio and ggplot2: an introduction

--- Summary

Create beautiful plots with RStudio and ggplot2's grammar of graphics

--- Description (HTML)

<p>
	The library's Digital Scholars Hub (DSH) is offering a free session designed for people who have used R and RStudio before but are new to plotting data with the 
	<strong>ggplot2</strong> package.
</p>
<p>
	During this hands-on session, you will:
</p>
<ul>
	<li>  Have a visualisation package installed (ggplot2)</li>
	<li>Learn about the basic ggplot2 components</li>
	<li>Use different kinds of visualisations</li>
	<li>Layer several visualisations</li>
	<li>Learn how to amend labels, themes and coordinates</li>
</ul>
<p>
	The software is available on the training room's computers. If you prefer to use your own laptop, you are also welcome to bring it in (but please install the software beforehand, or come 15 minutes earlier if you need to install the software).
</p>
<p>
	To install R and RStudio on your own laptop, refer to 
	<a href="https://gitlab.com/stragu/DSH/blob/master/R/Installation.md">our instructions</a>.
</p>
<p>
	It is recommended to attend the "R with RStudio" introductory session prior to this one, but not essential if you have used R before. You can review the 
	<a href="https://gitlab.com/stragu/DSH/blob/master/R/rstudio_intro/rstudio_intro.md">online material</a>, or follow the <a href="https://www.youtube.com/watch?v=BoDTNbP7_OQ">screencast</a>.<br>
</p>
<p>
	If you are interested in other R sessions, please search for "RStudio" on StudentHub or see the list of events on the Library's <a href="https://web.library.uq.edu.au/library-services/training">training page</a>.
</p>
