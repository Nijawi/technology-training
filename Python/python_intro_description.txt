--- Title

Python with Spyder: introduction to data science

--- Summary

Get started with Python and the Spyder IDE with this hands-on introductory session.

--- Details

<p>
	Python is a programming language that is widely used for scientific programming. The Library offers this session to get you started with using Python for data science.&nbsp;
</p>
<p>
	In this hands-on session directed at beginners, you will learn how to:
</p>
<ul>
	<li>create a project</li>
	<li>import useful modules</li>
	<li>create variables</li>
	<li>make use of functions</li>
	<li>differentiate attributes and methods</li>
	<li>import tabular data</li>
	<li>visualise data</li>
	<li>store code in a script</li>
</ul>
<p>
	We will use Spyder, an integrated development environment (IDE) for Python provided by the Anaconda distribution.
</p>
<p>
	The software is available on the Library computers, but you are welcome to use your own laptop. If you do, please make sure you install the Anaconda Python distribution 
	<em>before</em> coming to the session: <a href="https://www.anaconda.com/download/">https://www.anaconda.com/download/</a> (Python 3.x version)
</p>
